# 🌍 Overview

Tourist's Vision is an AI-powered tourism analytics platform developed as a graduation project by Data Science students at Umm Al-Qura University.

The platform transforms **1.2M+ tourism reviews** into actionable insights using **Arabic NLP, Machine Learning, Deep Learning, and Business Intelligence**, helping tourism stakeholders make data-driven decisions aligned with **Saudi Vision 2030**.

### Main Processing Steps

- Initial data cleaning and validation.
- Arabic text normalization.
- URL, emoji, and noise removal.
- Translation & language unification.
- Sentiment labeling.
- High-quality review filtering.
- Final dataset preparation for Machine Learning.
