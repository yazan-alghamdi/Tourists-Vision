# 📖 Overview

**Tourist's Vision** is an AI-powered tourism analytics platform developed as a graduation project by Data Science students at Umm Al-Qura University.

The project transforms over **1.2 million tourism reviews** collected from **Google Maps, TikTok, and YouTube** into actionable insights using **Arabic Natural Language Processing (NLP)**, **Machine Learning**, **Deep Learning**, and **Business Intelligence**.

It supports tourism stakeholders by measuring visitor satisfaction, identifying tourism trends, monitoring regional performance, and enabling data-driven decision-making aligned with **Saudi Vision 2030**.
