# 🌍 Tourist's Vision

An AI-Powered Tourism Analytics Platform for Sentiment Analysis of Tourism Reviews in Saudi Arabia

---

## 🚀 Project Highlights

- 📊 1.2M+ tourism reviews processed
- 🇸🇦 Coverage of all 13 Saudi regions
- 📍 428 tourist destinations
- 🌐 Google Maps, TikTok, and YouTube data
- 🤖 Arabic NLP + Machine Learning + Deep Learning
- 📈 Interactive Power BI Dashboards
- 🎯 Supports Saudi Vision 2030

---

## 📖 Overview

Tourist's Vision is a graduation project developed by a team of Data Science students at Umm Al-Qura University. The platform analyzes more than 1.2 million tourism reviews using Arabic NLP, Machine Learning, Deep Learning, and Business Intelligence to generate actionable insights for tourism stakeholders.
