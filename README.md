## 🌍 Overview

Tourist's Vision is an AI-powered tourism analytics platform developed as a Bachelor of Data Science graduation project at Umm Al-Qura University.

The platform analyzes more than **1.2 million tourism reviews** collected from **Google Maps, TikTok, and YouTube**, transforming large-scale Arabic tourism data into actionable insights using **Arabic NLP, Machine Learning, Deep Learning, Geospatial Analytics, and Business Intelligence**.

The project aims to support tourism authorities and decision-makers with intelligent dashboards, visitor sentiment analysis, destination benchmarking, and regional tourism insights aligned with **Saudi Vision 2030**.

## 📊 Dataset Summary

| Source | Reviews |
|--------|--------:|
| Google Maps | 1,016,595 |
| TikTok | 128,412 |
| YouTube | 73,555 |
| **Total** | **1,218,562** |

The dataset covers **428 tourism destinations** across all **13 regions of Saudi Arabia**.

---

## 🤖 Machine Learning Models

| Approach | Model | Purpose |
|---------|-------|---------|
| Traditional Machine Learning | LinearSVC | Fast and stable baseline for large-scale text classification |
| Deep Learning | Bi-LSTM | Captures sequential dependencies in Arabic reviews |
| Transformer | CAMeLBERT-DA | Context-aware Arabic dialect sentiment classification |

---

## 📈 Results

- Processed over **1.2 million** tourism reviews.
- Covered **428** tourist destinations.
- Included all **13 Saudi regions**.
- Built interactive **Power BI** dashboards.
- Compared **Machine Learning**, **Deep Learning**, and **Transformer-based** models.
- Achieved up to **99%** best experimental sentiment classification accuracy.
- Supported tourism insights aligned with **Saudi Vision 2030**.
